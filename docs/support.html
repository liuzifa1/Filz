Support

Help for Filz!, a native LocalSend-compatible file transfer app for iPhone and iPad.

Quick Help

Most transfer issues are caused by local network permissions, Wi-Fi isolation, or the receiving device not being visible on the same network.

  1. Make sure both devices are connected to the same local network.
  2. Allow Local Network access for Filz! in iOS Settings.
  3. Open Filz! on the receiving device and keep it awake during discovery.
  4. If discovery fails, use the app's option to add a client by IP address.

Sending Files

Use the Send tab to select files, photos, videos, or text. Choose a nearby LocalSend-compatible device when it appears, then confirm the transfer on the receiving device if required.

Receiving Files

Keep Filz! open while receiving. Received files can be opened from the app and saved to Files. If Save media to gallery is enabled, supported photos and videos can be saved to Photos.

Troubleshooting

Nearby devices do not appear

  • Confirm both devices are on the same Wi-Fi or local network.
  • Disable VPN, Private Relay, guest Wi-Fi isolation, or firewall rules that block local device discovery.
  • Restart the Filz! server from Settings, then try discovery again.

Transfers fail or stop

  • Keep both devices awake until the transfer completes.
  • Move both devices closer to the router or use a more stable network.
  • Try sending a smaller file first to confirm that discovery and transfer are working.

Received media is not in Photos

  • Enable Save media to gallery in Filz! Settings.
  • Allow Filz! permission to add items to your Photo Library.
  • Check the Files app if the item was received as a document instead of gallery media.

Privacy

Filz! does not collect, sell, share, or track user data. Files and text are transferred directly between devices on your local network. Read the full Privacy Policy.

Contact

If you still need help, contact the app developer through the support channel where you received Filz! Include your iOS version, Filz! version, device model, and a short description of what happened.
